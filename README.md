# ATEX Conference 2026 - Speaker Showcase

A premium "Show and Tell" internal pitch deck website for comparing speakers Jos Dirkx and James Taylor for ATEX Conference 2026.

## Features

- **Hero Section**: Split-screen display with speaker highlights and costs
- **Animated Counters**: Real-time animated statistics showing reach and influence
- **Interactive Comparison Matrix**: Side-by-side toggle between speakers with requirement fulfillment
- **Landed Cost Calculator**: Real-time AED cost calculator with adjustable travel parameters
- **Influence Map**: Visual UAE map showing client locations and partnerships
- **Framework Deep-Dive**: Interactive flip cards showing Human+ Era vs SuperCollaboration deliverables
- **Delegate Sentiment Chart**: Radar graph showing sector attraction by speaker
- **Content Previewer**: Video-heavy section with multimedia clips and technical rider info
- **Local Impact Ticker**: Scrolling banner with key metrics

## Tech Stack

- **React 19** with Vite
- **Tailwind CSS** for styling
- **Framer Motion** for animations
- **Chart.js** with react-chartjs-2 for data visualization
- **Lucide React** for icons

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
